# ProjectFlow — Project Manager App

A full-stack project management web application built with Next.js, MongoDB, and NextAuth.js.

## Live URL
https://project-manager-production-a983.up.railway.app

## Features
- Authentication with NextAuth.js
- Role-based Access Control (Admin and Member)
- Task Management with status tracking
- Dashboard with project overview
- REST APIs for tasks and projects

## Tech Stack
- Frontend: Next.js 16, React 19, TypeScript
- Database: MongoDB Atlas
- Auth: NextAuth.js
- Deployment: Railway

## Default Users
- Admin: admin@test.com / 123
- Member: member@test.com / 123

## Getting Started
npm install
npm run dev
